# Sea Organ

Status: L1 concept packet

This repository is a V5 L1 concept packet for a sea organ: a seawall or
shoreline installation that lets tide and wave motion push air through tuned
pipe chambers. It is a public-space wind instrument concept, not a
fabrication-ready marine structure or acoustic packet.

This packet contains no fabricated pipe dimensions, tuning values, bore
geometry, chamber volumes, pressure values, tide ranges, wave loads, seawall
details, CAD geometry, DXF coordinates, or measured validation data. All pipe,
air-path, wave-capture, drainage, marine-material, structural, and acoustic
claims remain `concept_only` or `pending_measurement`.

## Packet Map

- `design.md` - mechanism intent, open measurements, and L1 review questions.
- `bom.csv` - concept/pending-measurement subsystem families only.
- `visual-output-register.csv` - V5 authority register for the packet.
- `cad/mcp-session-log.md` - QMD and no-CAD/no-MCP provenance.
- `decision-record.md` - Step 0 result, decisions, open questions, and gates.
- `sourcing.csv`, `cut-list.csv`, `validation.csv`, `risks.md`,
  `drawing-brief.md` - V5 refresh baseline stubs; all TBD/pending_measurement,
  no site, pipe, marine, or CAD data yet.
- `evolution/` - evolution-pipeline Stage 0 intake: no CAD master exists yet,
  so this records the current concept-only intake only (Gate A not yet run).

## L1 Boundary

The next useful step is not a seawall build. It is a non-structural bench or
site-study plan that separates wave-air capture, drainage, saltwater
durability, pipe acoustics, public safety, and civil/shoreline engineering.
Any future L2/L3 packet needs measured site data and expert review before a
pipe layout, tuning table, marine structure, CAD file, or DXF release exists.
